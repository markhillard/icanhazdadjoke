## Requirements

[Node.js](https://nodejs.org/en/) `>=10.16.x`

## Usage

`npm install -g icanhazdadjoke`  
`icanhazdadjoke`

## Commands

`Enter`: get a random joke  
`q`: quit

## Options

`-version` (`-v`): show CLI version  
`-help` (`-h`): show CLI help
